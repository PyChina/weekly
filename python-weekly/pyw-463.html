Django数据库性能技巧 (59min) 大多数 Web 应用程序的性能问题都归结为一件事：数据库。在这个演讲中，Andrew Brookins 将帮助你从 Django 应用中榨取每一点的数据库性能。

用Python从公共API中获取新闻和文章 让我们来探索《New York Times》、《Guardian》、《HackerNews》和其他API，为你的下一个项目收集一些新闻数据吧!

如何将一个普通的gzip压缩包变成一个数据库 本文演示了如何将特别制作但普通的 gzip 归档文件用作类似数据库的存储。它还介绍了一个 Python 包，并解释了它的工作原理。

用TensorFlow 2.0进行时间序列预报 在本教程中，你将学习如何使用各种深度学习算法，如深度神经网络（DNN）、卷积神经网络（CNN）和循环神经网络（RNN），建立自己强大的时间序列预测模型。

Python Automation 101 （11min） 在本视频中，您将学习如何使用 Python 自动执行不同的操作！ 从像机器人这样的基于浏览器的 Instagram 开始，我们将逐步发展成为 Fortnite 机器人，该机器人可以在闲置模式下以 24/7 玩游戏。

使用NumPy和Rust进行优化 这是一篇关于用 NumPy 写的 Python 优化算法的快速文章，并在 Rust 中实现同样的代码。

常见的Python打包错误 概述了在创建和构建 Python 软件包时常见的错误以及如何避免这些错误。

如何将人类所有的知识装进一个盒子里 我们如何利用 Cython 的功能来帮助简化知识打包和共享的方式，即使没有互联网也是如此。

Constant Time LFU LFU 缓存驱逐方案最流行的实现，使用 min-heap，实现这三个操作的运行时间复杂度为 O(log n)，这使得 LFU 成为次优。在这篇文章中，我们将详细介绍一种巧妙的算法，它实现了 LFU，使所有的操作都以 O(1) 的运行时间复杂度发生。

有趣的项目、工具和库

minGPT 一个最小的 PyTorch 重新实现的 OpenAI GPT（Generative Pretrained Transformer） 训练。

fastMRI 用 AI 加速 MR 成像。Facebook AI Research（FAIR） 和纽约大学朗格尼健康中心的合作研究项目。目的是研究如何利用人工智能使 MRI 扫描速度提高10倍。

CODAR CODAR 是使用 PyTorch 构建的框架，用于分析帖子（文本/媒体）并预测帖子是否参与了网络欺凌和攻击。

computervision-recipes 计算机视觉的最佳实践，代码示例和文档。

mys Mys 编程语言--试图创建一种类似于 Python 的静态类型化语言，可以快速生成二进制文件。

present 一个基于终端的演示工具，具有色彩和特效。

volksdep volksdep 是一个开源工具箱，用于使用 TensorRT 部署和加速 PyTorch、Onnx 和 Tensorflow 模型。

Parth 启发式漏洞参数扫描仪。

django-openapi 在 Python2 + Django 中实现的一个最小的 FastAPI，无需 pydantic。

pe_tree Python 模块，用于使用 pefile 和 PyQt5 在树状视图中查看便携式可执行文件（PE）。也可以与IDA Pro一起使用，以转储内存中的 PE 文件和重建导入。

Minotaur A pythonic, asynchronous, inotify interface.

Databay 一个 Python 接口，用于预定的数据传输。它方便了在预定的时间间隔内将（任何）数据从 A 传输到 B。

handcalcs 用于将 Python 计算转换为渲染的 latex 的 Python 库。

Luminaire Luminaire 是一个 python 包，为监测时间序列数据提供 ML 驱动的解决方案。
