DockerizeMe：为Python代码片段自动推断环境依赖性。 DockerizeMe 是一种推断执行 Python 代码片段所需的依赖关系的技术，没有导入错误。 DockerizeMe 首先从 Python 包索引（PyPI）中离线获取流行的 Python 包的资源和依赖性知识。然后，它使用基于图的推理程序构建 Docker 规范。我们的推理过程解决了 Gistable 数据集中近 3000 个 gist 中 892 个 gist 的导入错误，Gistable 的基线方法无法找到并安装所有的依赖关系。

需要开始使用的高级SQLAlchemy功能 在 Python 中使用 SQLAlchemy 和它的混合属性、嵌套查询、表元数据、dialects 等可以轻松地使用 SQL。

在Django中使用自定义数据字段的多用户类型 学习如何用 Django 为多个用户类型添加自定义数据字段，以一种可维护的方式让我们轻松使用 Django 表单或 DRF 序列器。
