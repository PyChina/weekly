Tonic 一个创建开源自动驾驶系统及其硬件原型实现的项目。

fastapi-crudrouter 动态的 FastAPI 路由器，可以为你的模型自动创建 CRUD 路由。

Spectrum Spectrum 是使用机器学习生成 Rap 歌曲歌词的 AI。

CLIP 一个从自然语言监督中高效学习视觉概念的神经网络 CLIP 。它可以应用于任何视觉分类基准，只需提供要识别的视觉类别的名称，类似于GPT-2和3的 "zero shot "功能。

awesome-pattern-matching 以简单、强大、可扩展的方式实现 Python 3.8+ 的模式匹配。

briefcase Briefcase 是一个将 Python 项目转换为独立的本地应用程序的工具。你可以为 Mac、Windows、Linux、iPhone/iPad 和 Android 打包项目。计划支持 AppleTV、watchOS、wearOS 和 web 部署。

simple-graph 受 " SQLite 作为文档数据库 "的启发，在 SQLite 中建立了一个简单的图数据库。

Majel Mycroft 的前端，让你做令人兴奋的事情

svoice 具有多个扬声器的未知数量的语音分离。

pyHanko 用于 Python 的 PDF 签名工具包。

AutoGL 用于图上机器学习的 autoML 框架和工具包。

HyperTag 文件组织变得简单。HyperTag 让人类用标签直观地表达他们对文件的想法。

DnaChisel 一个 Python 库，用于根据一组约束和优化目标优化 DNA 序列。它也可以通过命令行界面或网络应用来使用。
