Visual Python Visual Python 是一个基于 GUI 的 Python 代码生成器，在 Jupyter Notebook 环境上作为扩展开发。

Tuplex Tuplex 是一个并行的大数据处理框架，以编译代码的速度运行用 Python 编写的数据科学管道。

plotext plotext 直接在终端上绘图，它没有依赖性，语法与 matplotlib 非常相似。它还提供了一个简单的命令行工具。

WikipediaQL 用于从维基百科有效提取数据的查询语言。

Blockman 一个 VSCode 扩展，用于突出显示嵌套的代码块，支持包括 Python 在内的多种语言。

Mava A library of multi-agent reinforcement learning components and systems.

Box Box 是一种基于文本的可视化编程语言，灵感来自虚幻引擎的蓝图功能图。
