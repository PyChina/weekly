  • Twitter 从 2011 起就用 PEX 方式将整个儿 Python virtualenv 打到一个 zip 中

pex 包括 Python 的包管理和分发, 原先是 Twitter 内部公用服务,已经开源为独立项目.

核心组件是和 .pex(Python EXecutable) 相关的 PEX 工具. 提供了类似 virtualenv 的通用虚拟环境.

Twitter 已规模使用了4年!

(是也乎:

电影: WTF is PEX? - YouTube

嗯哼,就是利用了 Python 内置的 zip 无缝解压能力, 将一切随时用 PEX 折腾到可命名/可执行/可升级/可部署的 .zip 文档中.
