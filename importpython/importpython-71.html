  • uvloop: 异常快的 Python 网络
    • web framework

asyncio 是当前 Python 的标准异步 I/O 网络框架. uvloop 则是能完全替代 asyncio 完备的更快的异步事务循环. 基于 libuv 以 Cython 构建; 事实上 uvloop 比 nodejs, gevent 或是其它异步框架快两倍以上. 性能接近 Go 语言实现的.
