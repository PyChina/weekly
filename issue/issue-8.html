Issue 8 ~ 蠎之禅

Hi Pythonistas.

首家赞助商已落实! 正如大家所知, MailChimp 不便宜, 特别是我们的订阅量增长这么快! 所以,我们必须采取措施确保收支平衡, 因此决定启动赞助机制. 赞助方,必须是在Python 社区有突出信誉的公司.

嵌入的广告必须是清雅的, 不显眼的, 内容必须完全由我们自行策划, 而且一直这样.

如果针对赞助商,还有什么问题/意见,请回复邮件说明.

RSS 和归档在 这儿 , 以及 这儿 .

成为巾帼或英雄, 作对的事儿, 要Follow @pycoders 如果你还不明白当期标题的含义, 进入命令行输入 

python -m this

Enjoy!

-- Mahdi and Mike

原文: Pycoder's Weekly (Issue #8) : Zen of Python

新闻与开发动态

  • Circus 0.2 发布ed Circus 是类似 supervisord 的进程监管工具, 现已发布0.1 版本, 项目很有意思,由 @tarek_zaide 以及 benoitc 创建

  • rauth 恨写一遍又一遍相同的OAuth的代码吧?! 这儿是来自 LITL 包含完全测试的模块, 目测,这事儿可以了了!

讨论

  • 字串重排的最佳方式?

这是在 rediit /r/python 频道引发的一个伟大的讨论, 有关对字符串排序以及重新的最好方式, 以及在 Python 中如何得到各种排列组合, 包含大量的神奇代码,相当深入的对 itertools 的使用经验, 值得关注.

  • Python中最困难的?! 过去十年, 没有任何讨论能同时引发新手以及专家一置的挫折以及好奇. 这就是 全局锁! 相关文章,此篇甚佳, 评注也异常有料!

项目

  • Django Sampler 引自仓库:"Django Sampler 允许对你的查询(SQL, Mongo,等)进行取样,给出慢执行. 并成组的指出在你的代码中哪里引发的查询. " 目前支持: Django SQL, Django Requests, Celery and MongoDB.

  • Melopy 玩弄声音的Python 库! 是对音乐发烧友梦想中的好物! 通过简单的代码,支持对声音播放/渲染 等等操作, 作出你自个儿的音乐!

  • DCPU16py 名为 0x10c 的工具,能在Notch的新游戏 中设计船舶, 虽然只能给每种船提供少数模拟资源, 但是 James Tauber 以前折腾过类似的模拟器,所以,这次很容易的整了出来.

文章

  • 回调是数据过程的抽象 有趣的文章, 讨论如何使用 JavaScript 中的回调方式, 在 Python( 具体是Twisted )中实现, 并使代码简洁,易读,好测试!

  • 对运行中的 Python 和 Django 进行标记以及追踪 文章就如何使用 trace 来追踪Django 的执行 给出了一些注意事项, 还通过 Django Plugin 提供了额外的管理命令, 来对 Django 项目进行执行跟踪!

  • Python HTTP的未来 Kenneth Reitz 探讨了Python 进行HTTP 响应的特性, 以及如何接入 Werkzeug . 未来可以共享的核心 HTTP 库, 已经启动有关仓库在 here .

  • Python中的计数 非常有趣的讨论, 有关如何在Python 的通用循环体中, 使用 range() 或 xrange(), 甚至执行体本身并不需要这种整数. 作者探讨了其它可实现相同目标的选择,逐一讨论, 对比.

  • Python objects, types, classes and Instances - 辞汇 继 Python 可调用的内因 之后, 作者对以往讨论的各种内部机制,进行了梳理, 重新说明了各种条目的含义, 对以往文章关注的朋友,这篇必须认真读哈!
