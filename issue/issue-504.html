(是也乎:

嗯哼? 这几乎就是自制语法糖了, 当年 UliWeb 中包含类似的自制模块;

from dataklasses import dataklass

@dataklass
class Coordinates:
    x: int
    y: int

然后就可以:

>>> a = Coordinates(2, 3)
>>> a
Coordinates(2, 3)
>>> a.x
2
>>> a.y
3
>>> b = Coordinates(2, 3)
>>> a == b
True
>>>
