项目

  • DissidentX (github.com)

Bram Cohen 发布的一种 Censorship resistance 工具,能将图形编码文件中.

  • MIDAS (github.com)

用以专发 Mac 上入侵分析系统的框架.

  • bamboo (github.com)

简化的实时数据分析应用. bammboo 支持通过界面对动态数据进行 合并,聚集,加入代数计算.

  • telegraphy (github.com)

Telegraphy 为 WSGI 应用提供了各种实时的附加功能, 例如: 事件过滤,订阅的持久化,授权/认证... 求试用!

  • timesched (github.com)

这是 daemontools 的工具接口. 能工作在 Windows 以及所有 Python 能跑的系统中, 包含完善的测试, 并不需要安装专用编译器.

  • timesched (github.com)

简洁的日历应用, 对于在 Python 中想使用 Angular.js 以及时区控制的,值得一试.

  • dejavu (github.com)

是否想知道 Shazam 怎么工作的? 音频指纹识别 算法已经在 Python 中实现!

  • 用Python 构建渐变色 (bensouthgate.com)

想知道怎么用 Python 以脚本形式来创建渐变色的话, 学习这篇带有示例代码的好文章吧!

  • 在Flask中 用Mock 和 Nose 进行测试 (safaribooksonline.com)

一篇伟大的教程, 教会你如何开始使用 Nose 测试 FLask 应用, 并有对 Mock 的良好介绍.

  • 改进你的Python: 理解单元测试 (jeffknupp.com)

梦幻般的好文章! 如果你在寻求测试水平的突破,从这儿开始!
