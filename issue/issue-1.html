Hi Pythonistas.

原文: Pycoder's Weekly (Issue #1) : Let there be light.

新闻与开发动态

  • Tornado 2.2

本次发布,对 Twisted的兼容 以及 WebSocket 的支持进行了高大升级

  • Django 1.4 Beta 1 发布

James Bennett, 已经宣告最新版本已经进入测试阶段. James 正式呼吁社区帮忙折腾,杀虫,以便在三月完成 1.4 版本的发布. 可以从 这儿. 获得最新clone, 嗯嗯嗯,他们已经不在对 Python2.4 进行测试了..

  • Mock 0.8

大家最爱的 模拟庉已经发布最新版本! Michael Foord 曰,这是一次非常巨大的变化, 并应允将通过 Blog 详细说明,

  • 最新 Pip and Virtualenv 发布

Carl Meyer, 宣布最新pip 以及 virtualenv将包含更多优秀工具.

  • Py3K 状态更新

Antonio Cuni, 一直以来孜孜不倦的支持 PyPy 成功迁移到 Python3 上. Antonio 刚刚更新了捐助列表,感谢大家对项目的支持!

  • Pypy 1.8 -- 照常发布

PyPy 的小伙伴们没有升级的要赶紧了! 成吨的bug 已经修复,太多更新以及内存效率的提升,是时候离开 1.7 了

讨论

  • SQLAlchemy Vs. Django ORM

此问题引发的一系列讨论收集在: /r/python 最初还比较 Naive, 但是,很快引发了有关 两者稳定性和鲁棒性的深入对比 Mike Bayer (SQLAlchemy's 作者) 介入讨论相当早,并给出了他的看法.

  • PyCon Web Development Summit 2012 Panel Discussions

如果你错过了今年 PyCon 的注册,现在可以通过此表单提交你的问题,并针对今年的讨论议题投票.

项目

  • Django-JHouston

"休斯敦,我们有一个Javascript的问题!" 这个项目能将浏览器中的JS 错误抓到数据库中,支持调试以及日志. 这对以往测试过程中难以追踪的JS 错误将非常有帮助. @pennersr

  • mitmproxy

mitmproxy 是支持 SLL 的中间件(man-in-the-middle-proxy). 支持 HTTP(等)协议在传输过程中进行编辑以及调试. 不仅仅是对路由的判定,允许你探知远程服务器接收到的是类型的数据! @cortesi

  • Django Pancake

关注性能的 Dj 模板. 存储展开后的模板, 在某些情况下比默认的模板能更快的执行. @adrianholovaty

  • PBS

PBS 是种特殊的子进程包装模块, 能将Python 的动态函式绑定到系统软件! 可以非常方便的将调用系统中的常见软件比如: ls, curl, cd 等等. PBS 提供了很干净的界面,帮助我们应用在脚本代码中.

文章

  • Running Sentry on DotCloud

Sentry 是开源实时软件日志聚合系统. Ken 提供了如何在 DotCloud 进行安装运行的全套教程. 只需要一步步照作就可以完成, 并提供了一个 DotCloud 的可部署仓库,来引导完成 Sentry 的安装. 感谢 Ken 的工作!

  • Advice for PyCon Speakers.

想在 PyCon 上演讲,首先要提升你的演讲水平! Eric Snow. 分享了其中的秘诀, 针对演讲的构思,幻灯的制作 等等方面提供了建议, 甚至于过程中,应该注意的以及避免的要点!

  • Virtualenvwrapper 3.0 with Python 3 support.

Virtualenvwrapper 是 Ian Bicking 的 virtualenv 的扩展工具集. 随着3.0 的发布,已经支持了 Py3, 这令大家感到, 社区在全力支持 Py3 的实用化.

  • Generator Tricks for System Programmers[2008].

由 David Beasley 贡献的教程, 探讨了 "生成器" 以及 "生成器"表达式在编程上下文中的各种技巧. 每一节都提供了现实世界真实的工程案例代码段.

  • Defining a WSGI App Deployment Standard

Tareq Zaide 将分享有关 WSGI 应用如何标准化以及他在 Mozilla 的实践. 俺是非常有兴趣在 PyCon 上 Tareq 的演讲的.

  • Pythons Valentines day gift to Clojure

嗯嗯嗯,空气中弥漫着浓浓的爱! Sung Ning 听说在 Ruby 社区已经在广泛的尝试 Clojure, 于是发布了开源工程: Pyclj 用Python 来读写 Clojure 的数据文本.

  • Python Performance Tips - Pt.1.

文章开头就吼,Python 的运行速度比不过那些编译型语言. 明显作者没听说过 PyPy. 不过,给出的一系列技巧,值得我们反复体验!

  • Python Web Framework Roundup

收集了很多 Python web 框架, 以及特性/发布日期等等描述, 统一使用 COOL 等级标注. 当然的, 少不了的'Hello World!' 的示例. Thanks Madis Väin!

  • Who's on Python 3?

是个伟大的仪表板, 可以关注前50大Python 项目对 Python 3 的支持程度. 你可以从 这儿 获得源代码! 由 Brett Cannon. 贡献!

  • Python Logging Best Practices

想对内置日志模块最细微处也能理解一,这绝对是篇好文章! Rob 还同时提供了幽默的阅读体验!
