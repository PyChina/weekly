    + GITHUB.COM/SNOACK

(`是也乎:`

人民如此想念 `goto` 不惜一切代价...
也要在 Python 中可以 goto.

![goto](https://ipic.zoomquiet.top/2021-09-22-ScreenShot%202021-09-22%2010.09.33.jpg)

```
from goto import with_goto

@with_goto
def range(start, stop):
    i = start
    result = []

    label .begin
    if i == stop:
        goto .end

    result.append(i)
    i += 1
    goto .begin

    label .end
    return result
```
